# WA10 - Sistema de Gestão e Planejamento

## 🚀 Sobre o Projeto
O **WA10** é uma solução desenvolvida para otimizar processos de comunicação e gestão de documentos, focando em uma experiência de usuário fluida e centralização de informações estratégicas.

Este repositório contém a implementação técnica do sistema, integrando as definições de negócio e design documentadas pela equipe.

## 🛠️ Tecnologias e Ferramentas
O projeto utiliza um stack moderno para garantir performance e escalabilidade:

* **Linguagens:** Java / JavaScript
* **Front-end:** HTML5 / CSS3 / Next.js
* **Back-end:** Java / SpringBoot
